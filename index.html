핵심 요구사항 분석

  • 마켓플레이스 양면 구조 — 브랜드사 제품 등록 → 글로벌 크리에이터 선택·홍보 → 리워드 정산까지 이어지는 양면 마켓 흐름 구현
  • 다국어 자동화 — 랜딩·앱·콘텐츠 전반의 다국어 자동번역, 정적 UI(i18n)와 동적 콘텐츠(번역 API) 이원화 처리
  • SNS 데이터 자동 수집 — Phyllo API 연동으로 SNS 팔로워 수 자동 수집 및 5단계 등급 자동 산정
  • 판매 추적 자동화 — TikTok Shop Affiliate·Shopify API 연동 판매 추적 링크 자동 생성
  • 글로벌 정산 자동화 — Payoneer Mass Payout API 연동 보상 정산 및 트랜잭션 무결성 보장
  • AI 기능 — Claude API 기반 실시간 번역 AI 상담(상담원 전환 포함), AI 숏폼 영상 재생 및 외부 영상 생성 API 연동
  • 실시간 처리 — Redis 기반 실시간 랭킹 탭 및 대량 트래픽 대비 캐싱
  • 멀티 콘솔 — 관리자(운영 전체)·판매자(조회 전용) PC 웹 분리 구축 및 스토어 등록·심사 대응

주요 화면 구상 (목업)

공고의 요구 기능과 글로벌 어필리에이트 플랫폼 벤치마킹(GRIN·Aspire·Impact.com·TikTok Shop·Levanta·Shopify Collabs)을 반영하여 핵심 화면 구조를 시각화한 와이어프레임입니다. 실제 디자인은 Phase 1 Figma 단계에서 제공 기획서·와이어프레임을 반영하여 최종 확정합니다.

제안 포인트

1. React Native 하이브리드 앱 + 멀티 플랫폼 동시 출시 경험. React Native로 iOS·Android·반응형 웹을 동시 출시한 프랜차이즈 관리 앱 수행 경험을 보유하고 있어, 단일 코드베이스로 양 스토어 심사까지 대응하는 본 프로젝트 구조에 즉시 투입 가능합니다.

2. 글로벌 다국어 대응이 가능한 팀 구성. 영어 네이티브(미국) 인력과 영어 구사가 가능한 베트남 인력을 보유하고 있어, 영문 UX 카피 검수·글로벌 크리에이터 대상 영문 상담 시나리오 설계·다국어 QA를 네이티브 수준으로 수행할 수 있습니다. 실제로 베트남 시장을 겨냥한 글로벌 네트워킹 플랫폼(Connectin)을 현지화 UX 기준으로 구축·운영한 경험이 있어, 단순 기계 번역을 넘어 글로벌 사용자에게 자연스러운 다국어 경험을 제공합니다.

3. 다수 외부 글로벌 API 통합 경험. 핀테크 플랫폼에서 7개 프로바이더·15개 이상 API 접점(본인인증·가상계좌·정산·SMS)을 단일 시스템에 연동한 경험과, 글로벌 네트워킹 플랫폼에서 OAuth·웹훅 기반 외부 슈퍼앱(Zalo)을 심층 연동한 경험을 보유합니다. Phyllo·TikTok·Shopify·Payoneer 등 10여 종 연동도 동일한 통합 패턴으로 안정적으로 처리합니다.

4. 글로벌 정산·트랜잭션 무결성 설계. 가상계좌 입출금·원리금 정산·은행 영업일 인식 정산을 자체 구현한 핀테크 경험을 바탕으로, Payoneer 정산의 멱등성(idempotency)·재시도·정합성 검증을 설계 단계부터 반영합니다.

5. AI 에이전트 자동화 + 관리자 대시보드 경험. 2026 「모두의 창업」 AI 솔루션 공급기업으로 선정된 AI 전문 역량을 보유하고 있습니다. 본 프로젝트의 핵심 배경인 "AI 에이전트 자동화"와 관련하여, Claude Agent SDK·MCP 기반 멀티 AI 에이전트 오케스트레이션 프레임워크를 직접 구축한 경험을 보유합니다. VC 펀드 관리 SaaS의 ChatGPT 연동 AI 보고서 자동 생성 및 200개 이상 관리자 API 경험과 결합하여, Claude API 실시간 번역 AI 상담·상담원 전환과 AI 성과 리포트, 관리자/판매자 대시보드를 검증된 패턴으로 제공합니다.

6. 우선순위 기반 단계별 납품. 산출물 완성도(1순위)와 일정 준수(2순위)를 동시에 달성하기 위해, 핵심 마켓플레이스·정산을 우선 납품하고 AI 숏폼·성과 리포트 등 부가 기능을 후속 단계로 배치하는 단계별 납품 구조를 제안합니다.

7. 글로벌 어필리에이트 플랫폼 벤치마킹 기반 설계. GRIN·Aspire·Impact.com·TikTok Shop·Levanta·Shopify Collabs 등 선도 어필리에이트/크리에이터 플랫폼을 분석하여 검증된 표준 패턴을 설계에 반영합니다. ① 크리에이터별 고유 추적 링크·코드 자동 발급·갱신, ② 등급별 차등 커미션 및 결제 시점 등급 스냅샷, ③ 14일 주기 정산 사이클(Payoneer), ④ 판매·전환·리뎀션 실시간 성과 대시보드, ⑤ AI 기반 자동 상담·성과 리포트 — 글로벌 사용자가 익숙한 UX와 정산 흐름으로 초기 신뢰도를 높입니다.

산정 근거

본 견적은 마켓플레이스 양면 구조, 10여 종 외부 API 통합 연동, 글로벌 정산, 다국어 자동화, AI 상담/숏폼 기능 등 전체 과업 범위를 기능 단위로 분해하여 산출하였습니다. 외부 글로벌 API 다수 연동 및 정산 무결성 검증 비중이 높아 백엔드 비중을 가장 크게 배정하였으며, 앱·관리자·판매자·랜딩 4개 프론트 영역의 동시 구축을 반영하였습니다. 산출물 완성도와 일정 준수를 우선하는 클라이언트 우선순위에 맞추어, 검증된 유사 프로젝트 수행 경험을 바탕으로 효율적 단계별 납품 구조로 설계하였습니다.

PRD (Product Requirements Document)

프로젝트 요구사항을 정의하고 기능 범위를 명확히 합니다.

1. 프로젝트 배경 및 목적

다국어 지원 및 AI 에이전트 자동화를 통해 글로벌 크리에이터가 언어 장벽 없이 브랜드 제품을 홍보할 수 있는 환경을 제공합니다. 브랜드사가 제품을 등록하면 글로벌 크리에이터가 이를 선택하여 홍보 활동을 진행하고 리워드를 정산받는 마켓플레이스를 구축하며, SNS 활동 데이터 자동 수집·판매 추적 링크 생성·글로벌 정산 자동화를 API 연동으로 매끄럽게 구현하는 것을 목표로 합니다.

2. 사용자 정의

사용자 유형설명주요 기능
